We have implemented this translation as an OCaml program called
\texttt{eo2lp}, which uses the Menhir parser generator for parsing
Eunoia signatures and proof scripts. The tool reads Eunoia files,
elaborates them using the operator from \autoref{sec:eo-elab},
applies the translation operators defined above, and outputs
corresponding $\lambdapi$ code.


\subsubsection{CPC-MINI.}
%
% table for CPC-MINI translation and checking.
% comparison to full CPC.
Because the full CPC signature is large and evolving, we created
a subset called CPC-MINI that covers the theories, inference rules,
and side-condition programs needed for the fragments of SMT-LIB
that we target.
%
CPC-MINI comprises 21 modules organized into theories
(e.g.\ \texttt{Builtin}, \texttt{Arith}, \texttt{Ints},
\texttt{Arrays}, \texttt{Sets}, \texttt{Quantifiers}),
inference rules (e.g.\ \texttt{Booleans}, \texttt{Arith},
\texttt{Uf}, \texttt{Rewrites}),
and side-condition programs
(e.g.\ \texttt{Nary}, \texttt{AciNorm}, \texttt{DistinctValues}).
%
All 21 modules are translated by \texttt{eo2lp} into $\lambdapi$
files that pass \texttt{lambdapi check} with full subject reduction
verification.
